# Restaurant-Reviews
A web-app to add or view reviews of restaurants. 
The restaurants can be filtered based on the following attributes - 
* Name
* Pin Code
* Cuisines

## Framework and Technologies used
### Frontend
* React JS
* HTML / CSS

### Backend
* Express JS
* Node JS

### Database
* MongoDB

### Dataset
* MongoDB - Sample_Restaurants
